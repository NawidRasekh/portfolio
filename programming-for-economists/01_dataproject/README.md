# Final Data Project

This folder contains the complete, structured data project.

## 📁 Structure

- `dataproject_final.ipynb` - Main notebook with all analysis
- `section2_inflation.py` - Python module with InflationAnalysis class
- `figures/` - Generated plots and visualizations

## 🚀 How to Run

1. Open `dataproject_final.ipynb` in VS Code
2. Make sure you have the required packages installed:
   ```bash
   pip install pandas numpy matplotlib requests
   ```
3. Run all cells: **Cmd+Shift+P** → "Notebook: Run All"
